もしもし

聞 き きましょう

I - 1 「もしもし」というのは、どんな意い味みですか。abcから１つ選えらびなさい。

[   a   b   c   ]

解答：a

I - 2 話の内容と合あっているものに○まる、違ちがっているものに×ばつをつけなさい。

a (   )     b (   )     c (   )    d (   )

解答：a (○)     b (×)     c (×)     d (○)

II もう一度聞いて、次の質問の答こたえを完かん成せいしなさい。

  1. 電話をかけるとき、日本語では、まずはじめに何といって相手に声をかけますか。

    2. まずはじめに「＿＿＿＿＿＿」と言って、声をかけます。

    解答：　もしもし

  2. この『もしもし』というのは、どんな意味がありますか。

    3. 「これから＿＿＿＿＿＿よ」という意味です。

    解答：　何か言いますよ

  3. 日本で電話が始まったのはいつですか。

    4. 今から＿＿＿＿＿＿の明治＿＿＿＿＿＿年です。

    解答：　１００年以上前　２３

ひとロメモ

「もしもし」は電話で使うときのほかに、人に呼よびかけるときにも使うことがあります。たとえば、忘わすれ物をして帰ろうとする人に、「もしもし、忘れ物ですよ」とか、物を落とした人に、「もしもし、カードが落おちましたよ」などと言って、呼びとめるときに使います。
